# README

This update includes compiled sequence data into one csv file with sequence data and available metadata. Priority lists have been split into 3 subcategories,
along with a top 100 most commonly seen "easy to pickup" species. Some of these may come as a surprise, but they have been flagged due to taxonomic changes
such as L. campestris which has split into 3 species, and should have all three of these sequenced before reassigning references to any of the potential
splits.
